HTML Questions and Answers

1. What is HTML and what does it stand for?

HTML stands for HyperText Markup Language. It is the standard language used to create and design documents on the web.

2. What is the difference between HTML and XHTML?

HTML is less strict than XHTML. XHTML must be well-formed, meaning all tags must be closed, nested properly, and in lowercase.

3. What are the basic building blocks of an HTML document?

The basic building blocks include elements like <!DOCTYPE html>, <html>, <head>, <title>, <body>, and various HTML tags that define content structure.

4. How do you create a hyperlink in HTML?

Use the <a> tag with the href attribute:

<a href="https://www.example.com">Click here</a>

5. What is the purpose of the "alt" attribute in the HTML "img" tag?

The alt attribute provides alternative text for an image if it cannot be displayed. It also improves accessibility for screen readers.

6. How can you create a table in HTML?

Use the <table>, <tr>, <td>, and <th> tags:

<table>
  <tr>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
  </tr>
</table>

7. What is the purpose of the "doctype" declaration in HTML?

The <!DOCTYPE html> declaration defines the document type and version of HTML being used, helping browsers render the page correctly.

8. What is the difference between HTML elements and HTML tags?

HTML tags are the markup used to define elements, while HTML elements include the tag and its content.

10. What is the significance of the "head" section in an HTML document?

The <head> section contains metadata, links to stylesheets, scripts, and other information that isn't displayed directly on the webpage.

11. How do you define headings in HTML?

Use the <h1> to <h6> tags for headings, with <h1> being the highest level and <h6> the lowest.

12. What is the role of the "title" element in HTML?

The <title> element sets the title of the document, which appears in the browser tab and is used by search engines.

13. How do you create an ordered list in HTML?

Use the <ol> tag for ordered lists and <li> for list items:

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

14. What are the different types of form elements in HTML?

Common form elements include <input>, <textarea>, <select>, <button>, and various types of input like text, checkbox, radio, etc.

15. How do you embed an image in an HTML document?

Use the <img> tag:

<img src="image.jpg" alt="Description">

16. What is the purpose of the "href" attribute in an anchor tag?

The href attribute specifies the URL of the page the link goes to.

17. How do you create a line break in HTML?

Use the <br> tag:

Line one.<br>Line two.

18. What is the purpose of the "target" attribute in an anchor tag?

The target attribute specifies where to open the linked document (e.g., _blank for a new tab).

19. What is the difference between block-level and inline elements in HTML?

Block-level elements take up the full width available and start on a new line, while inline elements only take up as much width as necessary and do not start on a new line.

20. How do you create a horizontal rule in HTML?

Use the <hr> tag:

<hr>

21. What is the role of the "span" element in HTML?

The <span> element is used to group inline elements for styling or manipulation without introducing a new block.

22. How do you create a checkbox in HTML?

Use the <input> tag with type "checkbox":

<input type="checkbox" id="check1">
<label for="check1">Check this box</label>

23. What is the purpose of the "src" attribute in the "script" tag?

The src attribute specifies the URL of an external script file to be included in the HTML document.

24. How do you create a dropdown menu in HTML?

Use the <select> tag with <option> tags:

<select>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>

25. What is the purpose of the "colspan" attribute in an HTML table?

The colspan attribute specifies the number of columns a cell should span in a table.

26. How do you add a background image to an HTML element?

Use CSS:

<div style="background-image: url('image.jpg');"></div>

27. What is the purpose of the "action" attribute in an HTML form?

The action attribute specifies the URL where the form data should be sent when submitted.

28. How do you add a video to an HTML document?

Use the <video> tag:

<video src="video.mp4" controls></video>

29. What is the purpose of the "disabled" attribute in an input field?

The disabled attribute makes an input field unmodifiable and non-interactive.

33. What is the purpose of the "required" attribute in an input field?
